# SPFX-Image-Mapper
SharePoint SPFX Web part for mapping an image with clickable sections

How it works:
* Add an image URL.
* Add Image mapping areas, and url

user clicks on the areas and is directed to url.

App uses <SVG> to create clickable sections.
